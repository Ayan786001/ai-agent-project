Prompt engineering handler om at designe instruktioner, som AI-modeller kan forstå.
Et godt prompt indeholder:
- Rolle (du er en ekspert…)
- Opgave (gør X)
- Format (returnér i JSON/markdown)
- Begrænsninger (max 300 ord, ingen kode)

I dette projekt bruges prompts til at lave syntese-rapporter.
