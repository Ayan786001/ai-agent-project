Qdrant er en open-source vektor-database, der bruges til semantisk søgning.
Man uploader embeddings (vektor-repræsentationer af tekst), som Qdrant indekserer.

Typisk flow:
1. Tekst → embedding
2. Embedding → Qdrant
3. Query → find lignende dokumenter
